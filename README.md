# Booking app

> Multi-tenant SaaS booking platform — Plateforme SaaS de réservation multi-tenant

## Stack

- **Frontend** — Nuxt 4, TailwindCSS v4, Pinia
- **Backend** — Nuxt 4 API routes (Nitro), Mongoose
- **Database** — MongoDB Atlas
- **Payment** — Stripe (optional)
- **Emails** — Resend + Vue Email
